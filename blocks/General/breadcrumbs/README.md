# Breadcrumbs Block

Блок для отображения хлебных крошек (breadcrumbs) на страницах сайта.

## Описание

Блок автоматически отображает навигационные хлебные крошки, используя класс `IH_Breadcrumbs` для генерации пути. Поддерживает два варианта цветовой темы: стандартную и золотую.

## Файлы блока

- `block.php` - Шаблон рендеринга блока
- `style.css` - Стили для фронтенда
- `editor.css` - Стили для редактора WordPress
- `script.js` - JavaScript функциональность блока
- `screenshot.png` - Скриншот блока для отображения в редакторе (опционально)
- `README.md` - Документация блока

## Настройки ACF

Блок использует следующие поля ACF:

- **breadcrumbs** (group)
  - **switcher** (select) - Выбор цветовой темы:
    - `default` - Стандартная тема
    - `golden` - Золотая тема

## Использование

1. Добавьте блок "Breadcrumbs Block" на страницу через редактор Gutenberg
2. Выберите цветовую тему в настройках блока
3. Блок автоматически отобразит хлебные крошки на основе структуры сайта

## Зависимости

Блок требует наличия класса `IH_Breadcrumbs` для работы. Если класс недоступен, блок не будет отображаться на фронтенде, но покажет предупреждение в режиме предпросмотра редактора.

## CSS Классы

- `.breadcrumbs-block-container` - Контейнер блока
- `.breadcrumbs-block` - Основной контейнер хлебных крошек
- `.breadcrumbs-block._golden` - Модификатор для золотой темы

## Структура HTML

```html
<div class="breadcrumbs-block-container _container">
    <div class="breadcrumbs-block [_golden]">
        <!-- Содержимое, сгенерированное IH_Breadcrumbs::render() -->
    </div>
</div>
```

## JavaScript функциональность

Файл `script.js` добавляет:
- Плавную прокрутку для якорных ссылок в хлебных крошках
- Улучшения доступности (ARIA атрибуты)
- Поддержку динамически загружаемого контента

## Скриншот блока

Для отображения кастомного изображения вместо иконки в редакторе WordPress, добавьте файл `screenshot.png` (рекомендуемый размер: 120x120px) в папку блока. Если файл отсутствует, будет использована стандартная иконка на основе категории блока.

## Кастомизация

Стили блока можно переопределить через файл `style.css` или через основной файл стилей темы.
